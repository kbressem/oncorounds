# Changelog

## 1.0.0 (2025-02-20)

Initial public release.

- Three-round interactive benchmark for clinical oncology reasoning
- 20 expert-curated oncology cases
- OpenRouter client supporting 200+ models
- OpenAI client for direct API access
- Automated scoring across diagnosis, differentials, and treatment
- Resume capability for interrupted runs
- JSON schema validation for all inputs and outputs
- `oncorounds-score` CLI for scoring benchmark outputs
